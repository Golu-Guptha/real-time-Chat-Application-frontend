# Mini Team Chat Application

A real-time team chat application built with MERN stack (MongoDB, Express, React, Node.js) and Socket.io.

## Features
- Real-time messaging
- Channel management (Create, Join, List)
- User Authentication (JWT)
- Online Presence
- Message History

## Tech Stack
- **Frontend**: React, Vite, Socket.io-client, Axios
- **Backend**: Node.js, Express, Mongoose, Socket.io, JWT
- **Database**: MongoDB Atlas

## Getting Started

### Backend
1. `cd backend`
2. `npm install`
3. Configure `.env` (see `.env.example` or create one with `MONGODB_URI`)
4. `node server.js`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`

## Deployment
- Backend can be deployed to Render/Railway/Fly.io.
- Frontend can be deployed to Vercel/Netlify.
